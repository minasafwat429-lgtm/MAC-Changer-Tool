# MAC Changer

A simple Python tool that allows users to change their MAC address on Linux systems.

## Features
- Change MAC address easily
- Verify current MAC address
- Simple command-line interface

## Requirements
- Python 3
- Linux system
- root privileges

## Installation

Clone the repository:

git clone https://github.com/minasafwat429-lgtm/Mac_Changer.git

Navigate to the folder:

cd Mac_Changer

## Usage

Run the script with root privileges:

sudo python3 Mac_Changer.py -i eth0 -m 00:11:22:33:44:55

Example:

sudo python3 Mac_Changer.py -i wlan0 -m 00:AA:BB:CC:DD:EE
